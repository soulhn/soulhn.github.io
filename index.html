BUILD 01 — 2026.07 리빙 포트폴리오 — soulhn.github.io 컨텍스트 엔지니어링

단일 HTML + 바닐라 JS, 프레임워크 없이 기획→빌드→배포. 상태 없는 정적 페이지에 프레임워크는 과잉이라 판단 — 인터랙티브 레이더까지 바닐라로. 콘텐츠는 미리 구조화해둔 커리어 데이터를 AI가 읽어 생성.

빌드 리포트 열기

문제 — 증거(대시보드·repo·블로그)가 흩어져 있어, 나를 3분 안에 파악할 단일 URL이 없었다.

AI에게 맡긴 것 — 코드 전량(레이더 SVG·인터랙션 포함), 콘텐츠 초안, 배포 자동화.

직접 판단한 것 — 콘셉트 전환(역량 지도·라이트 테마), 공개 범위와 표현의 수위, 스코프 절단 — 방향 수정 약 20회.

검증 — 완료 기준 5개를 먼저 확정하고, 수정마다 실물 확인 + 전수 검사. 5/5 달성, 기획→배포 2시간 11분.

Q. 왜 프레임워크를 안 썼나 — 상태 없는 정적 페이지에 프레임워크는 과잉. 도구는 문제 크기에 맞게 고른다.

Q. AI가 다 한 것 아닌가 — 타이핑은 AI가 다 했다. 문제 정의, 완료 기준, 무엇을 공개하지 않을지, 그리고 20회의 방향 수정이 사람의 몫이었다.
